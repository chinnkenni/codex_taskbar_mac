Codex Taskbar v0.1.4 修复 Codex 新版本下无法读取当前正确额度的问题。

## 修复

- Codex 0.140.0 之后，当前额度从旧的 `codex.rate_limits` websocket 事件迁移到了 HTTP 响应头。
- App 现在会优先读取新的 `x-codex-primary-*` 和 `x-codex-secondary-*` 响应头。
- 继续兼容旧版 `codex.rate_limits` 事件。
- 修复构建脚本在 `Documents` 目录下可能因为 macOS 扩展属性导致签名失败的问题。

## 当前显示逻辑

- `5h` 显示 5 小时窗口剩余额度和具体重置时间。
- `1w` 显示 1 周窗口剩余额度和重置日期。

## 下载安装

下载下方的 `Codex-Taskbar.dmg`，打开后把 `Codex Taskbar` 拖到 `Applications` 即可。

## 安装安全提示

当前版本未经过 Apple 公证。如果首次打开时弹出“Apple 无法验证是否包含恶意软件”的提示，可以在系统设置的 `隐私与安全性` 中点击 `仍要打开`。详细图文流程见 README。

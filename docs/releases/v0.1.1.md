Codex Taskbar v0.1.1 是安装包修复和版本机制更新。

## 修复

- 修复 Release 中 DMG 安装后可能提示“应用已损坏”的问题。
- 打包流程改为在干净临时目录中清理扩展属性、重新签名、严格校验后再生成 DMG。

## 新增

- 新增 `VERSION` 文件统一管理版本号。
- App 菜单中显示当前版本。
- App 支持检查 GitHub Releases 最新版本。
- App 启动后每天自动检查一次更新。
- 菜单中新增 `检查更新`，发现新版时可直接打开最新 Release 页面。

## 下载安装

下载下方的 `Codex-Taskbar.dmg`，打开后把 `Codex Taskbar` 拖到 `Applications` 即可。
